while文

while文はfor文同様繰り返しを実現する。

さっそくだけどwhile文の構文を以下に示す。

            while(条件){
	            繰り返す処理
			}

条件が真な限り処理を繰り返すよ。

for文の初期処理と繰り返し後の処理を書かない場合、 

            for(; 条件;){
	            繰り返す処理
			}

これと同じだと考えればいいよ。

for文とwhile文の使い分けだけど、ここは英語の意味になって・・

forは「～から～」って感じだから繰り返す範囲や回数が決まっていたりする時。

whileは「～の間」って感じだから回数とか決まってなくてある条件の間って感じの時。

まぁ、行いたいことが実現できればどっちを使おうなんて気にしなくていいと思うよ。



注意してほしいこと

while文は繰り返しが終了しない無限ループに陥る可能性がfor文より高いように思う。

例えば以下のコードを見てほしい。

		    int a = 0;
            while( a<30 ){
	            System.out.println(a);
			}

これはいつまでたってもa<30は真のままだから永遠にaの値を出力させ続けるよ。

これを解決するには、

		    int a = 0;
            while( a<30 ){
	            System.out.println(a);
	            a++;
			}

こうすればいいね。参考程度にこれをfor文で書き換えると、

            for(int a = 0; a<30; a++){
                System.out.prinltn(a);
            }

こうなるよね。
