インクリメントとデクリメント

よく目にするから説明を載せておくよ。

インクリメントってなんだ

インクリメントってなんだって書いておきながら、まずはある変数iがあるとしてこのiが持っている値を1だけ増加させる方法を考えよう。

iに5が入っていたのを6、iに10が入っていたのを11にするって感じなんだけどこれは以下の様なコードで実現できるよね。

					i = i+1;

こんなのだったり

					i += 1;

こんなのだったりね。インクリメント演算子は上の様な1だけ値を増加させる時に使えるものだよ。（三種類目の書き方みたいな感じかな）

じゃあ、実際どのようにインクリメント演算子を使うか例を上げるね。

					i++;
					++i;

例の一行目も二行目もどちらも「iの値を1増加」させているよ。

++変数名か変数名++とすることで実現が可能なんだね。

注意してほしいのは、

					i++;
					++i;
					i = i+1;
					i += 1;

上の4つは同じ意味だけど、これら4つと

					i + 1;

は同じ意味にはならないことだよ。代入しているか代入してないかという点が異なっているよ。



デクリメントってなんだ

デクリメントは「値を1減少させる」ものだよ。

					i--;
					--i;

例の様に使用することができるよ。++が--になっただけ。



例

			class Test{
				public static void main(String[] args){
					int i = 0;
					System.out.println(i);
					i++;
					System.out.println(i);
					++i;
					System.out.println(i);
					i--;
					System.out.println(i);
				}
			}
		
			0
			1
			2
			1


i++と++iの違い

					i++;
					++i;

は、どちらも1増加させるって意味だったね。実はこの二つには違いがあるんだ。

これまでのように++iや--iを単体のみで使用している分には違いは生まれない。

ただし、これらを他のもの（イコールとかね）と一緒に使うと違いが生まれえることがある。どういうことかというと

			class Test{
				public static void main(String[] args){
					int a = 0;
					int b = 0;
					b = a++;
					System.out.println(a);//1を出力
					System.out.println(b);//0を出力
				}
			}

と

			class Test{
				public static void main(String[] args){
					int a = 0;
					int b = 0;
					b = ++a;
					System.out.println(a);//1を出力
					System.out.println(b);//1を出力
				}
			}

では結果が異なるってことだよ。違いを生んでいるのは5行目だね。（そこしか違いないし）

例1-1の

					b = a++;

の部分は

					b = a;
					a++;

という流れで処理されていると考えればいいよ。まず、bにaの値が代入されてその後にaの値が1増加されるって感じ。

逆に例1-2の

					b = ++a;

の部分は

					a++;
					b = a;

という流れで処理されていると考えればいいよ。まず、aの値が1増加されてその後にbの値がaに代入されるって感じ。

つまり、i++とした場合はiの値が1増加されるのが後からになるってこと。このことを示す例をいくつかあげておくね。

			class Test{
				public static void main(String[] args){
					int b = 0;
					System.out.println(b++);//0が表示される。表示の後に1増加する。
					System.out.println(b);//1を表示
					System.out.println(++b);//1増加して2になる。2を表示。
				}
			}
		
			class Test{
				public static void main(String[] args){
					int b = 0;
					System.out.println(b++ * 10);//0*10の0が表示される。表示の後に1増加する。
					System.out.println(b);//1を表示
				}
			}

こんな感じ。

--iとi--の違いも同じだからね。
