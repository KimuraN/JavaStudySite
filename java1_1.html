Javaまずはここから

まずはこれから

         class Test{
        
    	    public static void main(String args[]){
    		    System.out.println("Hello");
    	    }
    
        }

とりあえず、Test.javaファイルを作って上の様なコードを書いて実行してみて。

え?ファイルの作り方がわからない？？

emacs Test.javaとターミルにコマンドを打てば良いよ。

Helloと表示されたはず。おめでとう。それじゃあ、ここからjavaの説明をしていくよ。



１つ以上のクラスが必要

Javaのプログラムはクラスがいくつか集まってできている。つまり1つ以上クラスが絶対必要ってこと。

クラスってなんだよって思うだろう。しかし、クラスについてはのちのち詳しく説明するから、 今はjavaプログラムを作るものみたいに考えておいてほしい。

クラスは以下の様なコードで表すことができる。

  class クラス名{



  }

上の例1ではTestというクラス名のクラスを用意している。

このクラスの中に色々に色々書き加えていくよ。

クラスの中ってどこだよっていうと｛と｝で囲まれた部分。ちなみに｛と｝で囲まれた部分をブロックというよ。

ルールとして原則クラス名とファイル名は同じじゃないとダメ。上の例でもクラス名がTestだからTest.javaってなってるでしょ。つまり以下に述べるようなのはダメだよ。

	class Test{

	}
  
	class TEST{

	}

最後になったけど、クラス名の先頭は大文字にするっていう暗黙のルールがあるからとりあえず大文字にしておこう。



Mainメソッドが実行される。

javaにはクラスが必要って言ったけど、クラスだけは実行できないよ。(コンパイルはできる)

なぜなら、計算機はmainメソッドの中に書かれた内容を上から順に実行していくからだよ。

mainメソッドとは上の例でいう

    	public static void main(String[] args){
		    //ここに書いたものを上から実行していく。
	    }

の部分だよ。

とりあえずはmainメソッドの中に書いたもの上から順に実行されることが分かれば良いよ。

mainメソッドというものの中ってどこかというとやっぱり｛と｝で囲まれた部分。

つまり、下の様なものはダメ。

    	class Test{
	    	public static void main(String[] args){

		    }
		    System.out.println("Hello");
	 }

javaにはクラスが必要だから、mainメソッドだけ書いた以下の様なコードももちろんダメだ。

		public static void main(String[] args){

		}


文字の出力

文字の出力は、

	    System.out.println(出力させたい文字列);

の様にして行うことができる。上の例では、

        System.out.println("Hello");

となっていたから、Helloと表示される。文字列は""で囲む必要があるため、Helloでなく"Hello"とする必要がある。

文字列以外にも色々なものを表示できるよ。

        System.out.println("Hello"  + "こんにちは");
    
        System.out.println(3+4);
    
        System.out.println("3+4");
    
        System.out.println("Hello"+ 3 + 4);

まず、"Hello"+3で"Hello3"となって、"Hello3"+4で"Hello34"となる。

        System.out.println("Hello"+ (3 + 4));

まず、(3+7)で7となり、"Hello"+7で"Hello7"となる。これについては優先順位を説明をみると 分かるようになると思うから、今なぜこうなるのか分からなくても大丈夫。
