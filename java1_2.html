コメント

コメントはとっても便利だからさっそく紹介しておくよ。使わないと損するよ。

コメントってなんだ

コメントとはコードの説明をするものだと考えてくれれば良いよ。

人間のために書いた説明を計算機が読む必要はないからコメント部分を計算機は無視するよ。



コメントを使おう

以下に実際にコメントを使っている例をあげて説明する。

		class Test{
		
			//これはmainメソッド
			public static void main(String[] args){
				/*
				したに書いてあるものはtestと出力する。
				コメントテスト
				*/
				system.out.println("test");
			}
		
		}

上の例では二種類のコメントを使っている。以下で説明するね。

			//これはmainメソッド

この部分が一個目のコメントだよ。//の後ろの記述をコメントとするよ。

ただし、//の後ろの記述しかコメントとならないから複数行のコメントは無理だよ。

		/*
		したに書いてあるものはtestと出力する。
        コメントテスト
        */

この部分が二個目のコメント。/*を記述後に*/がくるまでの全ての記述をコメントとするよ。

*/がくるまでコメントとするから複数行のコメントが可能だよ。

ちなみに、コメントは色々なところに書くことができるから、

		class Test {

			public static/*コメント*/ void main/*コメント*/(String[] args){
				System.out.
				/*
				コメント
				*/println(/*コメント*/3);//コメント
			}

		}

こんなこともできるけど、こんな見にくいのはダメダメだから。

どうでも良いことやわかりきったことをコメントしている最初の例もダメダメなコメントだからね。



一時的に処理を無効化しよう

有用なコメントの使いかたを紹介しておくよ。

		class Test{

			public static void main(String[] args){
				system.out.println("test");
				system.out.println("aka");
			}
		
		}

上のコードの5行目の処理を一時的に行わないようにしたい。 system.out.println("aka");を消せば処理は行われなくなるのだが、これはちょっと躊躇われる。 だって一時的が終ればまた必要になってsystem.out.println("aka");を書きなすはめになることが予想されるからね。

この問題はコメントを使い解決できる。

		class Test{

			public static void main(String[] args){
				system.out.println("test");
				//system.out.println("aka");
			}
		
		}

上の様に5行目をコメントとすれば良いんだね。コメント部分を計算機は無視するからこれで5行目は処理されなくなるよ。もとに戻したい時は//をはずすだけだしね。
