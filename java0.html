Javaプログラムの実行の仕方

書いたプログラムを実行する方法を説明する。

例をあげた方が分かりやすいから、Test.javaというファイルを作ったとして、これを実行したいとしよう。


1．コンパイルする

コードが書き終わって早速実行したいところだけど、その前にコンパイルが必要なんだ。 コンパイルとはプログラム言語で書いたプログラムを、 計算機が実行できる形(オブジェクトコード)に変換することだよ。

	javac Test.java

上のコマンドでコンパイルできる。コンパイルが終了するとTest.classというファイルが作られるよ。
これがオブジェクトコード。

javac　ファイル名.java

で、ファイル名.javaをコンパイルしてファイル名.classというオブジェクトコードができるって感じ。



2. 実行する

1.で計算機が実行できる形に変換できたので、いよいよ計算機に実行させよう。

	java Test

上のコマンドを実行するとプログラムを実行できる。

Test.javaはこんちはと表示するプログラムなので、実行するとこんちはと表示されたはず。

java　ファイル名

で、実行するって感じ。コンパイル時と違って.javaは必要ないよ。(なぜなら本当は、java クラス名　って感じだから)
