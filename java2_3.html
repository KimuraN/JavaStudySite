boolean

intとかcharとかと違ってbooleanちょっとよくわかんないかもしれないから補足しておくよ。Cにはなかったしね。

後に説明する条件式と組み合わせてつかうことが多くなるよ。

trueとfalse

boolean型の値はtrueとfalseだけしかない。

つまりboolean型の変数を宣言したとしたらその変数に代入できるものはtrueかfalseのどちらかってこと。

			class Test{
				public static void main(String[] args){
					boolean b = true;
					boolean b2 = false;
				}
			}

上のコードような感じでね。これはあまり面白くないけど、以下の様なコードも可能なんだよ。

			class Test{
				public static void main(String[] args){
					boolean b;
					b = 1<10;
				}
			}

例2ではbに1<10を代入しているかの様にみえるね。でもbには上でも述べた様に、trueかfalseしか入らないよ。

1<10は真か偽かっていったら真だよね。つまりbにはtrueが代入されるよ。 疑う人はSystem.out.println(b)ってやってごらんよ。

同じような例をさらに示すね。

			class Test{
				public static void main(String[] args){
					boolean b = ( 0>1 );//falseが代入
					boolean b1 = ( 1<=30 );//trueが代入
					boolean b2 = ( 1<1 );//falseが代入
					boolean b3 = ( 1<=1 );//trueが代入
					boolean b4 = ( 1==1 );//trueが代入
					boolean b5 = ( 1!=8 );//trueが代入
					boolean b6 = ( 1!=1 );//falseが代入
				}
			}

例3のうちみなれないものを説明しておくね。

==は右辺と左辺が同じかを調べるよ。同じだったらtrue、異なっていたらfalseだね。

!=は右辺と左辺が異なっているかを調べるよ。同じだったらfalse、異なっていたらtrueだね。

ちなみに!はノットみたいな感じで使うことができるよ。これも例を示すね。

			class Test{
				public static void main(String[] args){
					boolean b = !true;//falseが代入
					boolean b1 = !( 1<=30 );//falseが代入
					boolean b2 = !( 1!=1 );//trueが代入
				}
			}

かつとまたはも使うことができるよ。trueかつfalseはfalse、trueまたはfalseはtrueみたいな感じでね。これも例を示すね。

かつが&&、またはが||となっていると考えてくれればいいよ。

			class Test{
				public static void main(String[] args){
					boolean b = ( 1<20 ) && ( 2<30 );//trueが代入
					boolean b1 = ( 1<=30 ) && ( 3<-30 );//falseが代入
					boolean b2 = ( 1<=30 ) || ( 3<-30 );//trueが代入
					boolean b3 = ( true || false );//trueが代入
					boolean b4 = ( true && true );//trueが代入
					boolean b5 = b && b1;//falseが代入
				}
			}

注意してほしいのは例えば3<2を例とすると(3<2)自体がfalseなのではなく、(3<2)を演算した結果falseが得られるということ。あんまりわかんないよね。 int i = 2+8;とするとiの値は10になるけど、それは(2+8)自体が10だからではなく(2+8)を演算した結果10が得られてiが10になるのと同じこと。これで わかりやすくなったかな。
