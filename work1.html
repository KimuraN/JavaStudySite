フレーム

フレームってのはwindowsでいうウィンドウ。ブラウザとかゲームとか起動させると画面が表示されるよね。

その画面だと思えばいいよ。

        import javax.swing.JFrame;
        
        public class Test{
            
            public static void main(String[] args){
                JFrame frame = new JFrame("FrameName");
                frame.setSize(640,480);
                frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
                frame.setVisible(true);
            
            }
        
        }

例を実行してごらん。フレームが表示されたよね。おめでとう。

説明

        JFrame frame = new JFrame("FrameName");

JFrame型の変数frameにJFrameを生成。 引数に文字列を与えるとその文字列がフレームの名前になるよ。

        frame.setSize(640,480);

フレームのサイズを決めるよ。幅、高さの順でint値を引数に与えるよ。

        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

フレームを閉じた時にどう処理するか決めているよ。×ボタンを押したらどうするかってこと。

今回はフレームを閉じたらプログラムも終了するようにしたよ。

            frame.setVisible(true);

フレームが見えるか見えないか。つまり表示するかしないかを決めるよ。引数にtrueを 与えているからフレームを表示させるよ。falseなら表示されないよ。
