switch文

switch文はif文同様にある条件に応じた処理を行うもの。

例えばif文を用いて以下の様なコードを書いたとしよう。

    			int i = 20;
    			
	    		if( i==0 ){
	    			System.out.println("0");
	    		}else if( i==10 ){
	    			System.out.println("10");
	    		}else if( i==20 ){
	    			System.out.println("20");
	    		}else if( i==30 ){
	    			System.out.println("30");
	    		}else{
	    			System.out.println("else");
	    		}

else ifが続いて面倒だね。これをswitch文を用いたコードに書き換えると、

    			int i = 20;
    			
	    		switch( i ){
	    			case 0:
	    				System.out.println("0");
	    				break;
	    			case 10:
	    				System.out.println("10");
	    				break;
	    			case 20:
	    				System.out.println("20");
	    				break;
	    			case 30:
	    				System.out.println("30");
	    				break;
	    			default:
	    				System.out.println("else");
	    				break;
	    		}

ちょっとだけスマートになったような。switch文の構文を以下に示すね。

	    		switch( 式 ){
	    			case 値1:
	    				式の価が値1と一致した時の処理
	    				break;
	    			case 値2:
	    				式の値が値2と一致した時の処理
	    				break;
	    			case 値3:
	    				式の値が値3と一致した時の処理
	    				break;
	    			case 値4:
	    				式の値が値4と一致した時の処理
	    				break;
	    			default:
	    				式の値が値1から値4のどれとも一致しなかった時の処理
	    				break;
	    		}

式の部分の値と一致したところの処理が実行されるって感じ。 defaultは一致する値がなかった時にされる処理。defaultは書かなくてもいいよ。 一致する値がなかったら何もしないって感じでね。

上の例1を例に再度説明するね。式の部分がiとなっている。iは20となっているため、 この値と一致している部分の処理、すなわち20と表示される処理が行われるって感じ。

breakについて触れていないけどこれは後に説明するから今は無視して。
