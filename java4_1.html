配列

配列とは

例えばint型の変数を100000個用意したいとしよう。

        int a1 = 1;
        int a2 = 2;
        int a3 = 3;
        ・・・・
        int a31034 = 31034;
        int a32035 = 31035;
        ・・・・

これは面倒だね。

配列はこの様な同じ型の 変数を複数利用したい時に使うと便利。たくさんの並んだ箱だと考えればいいよ。

実際の使い方は以下で説明するよ。



配列の宣言

変数を用意する時は型と変数名が必要だったね。

        int num;

の様にね。それでは、配列の場合はどうするのか。

[]をつけるだけで良いんだよ。どういうことかって？？

        int[] is;
        char[] cs;

この様にすると、 int型の配列の変数isとchar型の配列の変数csということになる。

ちなみに、

        int is[];
        char cs[];

の様に、変数名の後ろに[]をつけても同じ意味になるよ。



配列の生成

変数を用意したら配列をつくらないとね。配列をつくる時は初めに配列の大きさを 決める必要がある。

配列の大きさってのはいくつの連続した箱かってこと。

ちなみに後から配列の大きさを変えることはできないからね。それじゃあ、下に例を示すね。

        is = new int[10];

これは大きさが10の配列を作った例である。

        変数名 = new 型[配列の大きさ];

って感じだね。このことを配列を生成するというよ。生成した配列には10 個の箱があるけど中には何も入っていないよ。次は値を代入してみよう。 

        is[0] = 1;
        is[1] = 2;
        is[2] = 3;
        ・・・
        is[9] = 10;

こんな感じだね。

        変数名[添え字]

上の様に添え字を使って何番目の箱かを指定することができるよ。

ちなみに添え字は0始まりだから9までだよ。

宣言と生成を同時にすると、

            int[] array = new int[10];
            String[] array2 = new String[3];

配列を生成した後にそれぞれの要素を代入するのは面倒だなって時には、

            int[] array = {1,2,3}
            boolean[] array2 = {true,true};

の様な記述も可能だよ。これは、 

            int array = new int[3];
            array[0] = 1;
            array[1] = 2;
            array[3] = 3;
            boolean array2 = new boolean[2];
            array2[0] = true;
            array2[1] = true;

と同様の意味だよ。

ただし上の記述を使う時は変数の宣言と同時に行う 必要があるよ。変数だけ用意しておいて・・・ってのはだめってこと。

ダメな例を以下に示しておく。

        int[] array;
        array = {1,2,3};

例

        class Test{
            
            public static void main(String[] args){
                int array = new int[3];
                for(int i = 0; i < array.length; i++){
                    array[i] = i;
                }
                for(int i = 0; i < array.length; i++){
                    System.out.println(i);
                }
            }
        
        }

1,2,3が出力される。配列名.lengthで配列の大きさを獲得できるよ。今回の場合は3だね。

        class Test{
            
            public static void main(String[] args){
                int array = {1,2,3};
                for(int i = 0; i < array.length; i++){
                    array[i] = array[i]*2;
                }
                for(int i = 0; i < array.length; i++){
                    System.out.println(i);
                }
            }
        
        }

2,4,6が出力される。
