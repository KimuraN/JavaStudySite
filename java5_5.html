犬の生成



犬が生まれるよ

5-4で作成したクラスは

        class Dog{
            String name;
            int age;
        
            public void bark(){
                System.out.println("wan wan");
            }
            
            public void run(){
                System.out.println("running...");
            }
        }

こんなんだったね。

クラスは設計図と例えたように、これは犬ではない。犬をつくるための設計図である。 よっていまからこのクラスから実態がある犬を生成する。

Dogクラスはmainメソッドを持っていないため、犬を生成し実行するためのTestクラスを以下の 様に作成する。(Test.javaという新しいファイルを作るんだよ) 

        class Test{
 
            public static void main(String[] args){
            
            }
            
        }

まず生成した犬をいれるための変数を用意する。int aみたいな感じだったよね。ただデータの種類はintでもstringでもない。

データの型はDogである。クラス名と同じ。変数を用意するとこんな感じ。

        class Test{
 
            public static void main(String[] args){
                Dog mydog;
            }
            
        }

Dog型のmydogという変数名の変数を宣言した。

この時点で、mydogの中は何も入っていないよ。整数の時だって、int aとしただけでは aには何もはいってないでしょ。

では実際に、mydogにDogクラスから生成した実態を入れると

        class Test{
 
            public static void main(String[] args){
                Dog mydog;
                mydog = new Dog();
            }
            
        }

この様になる。実態のことをインスタンスと呼ぶから、この一連の 処理をインスタンスの生成と言うよ。今の処理を一度に行うと、 

        class Test{
 
            public static void main(String[] args){
                Dog mydog = new Dog();
            }
            
        }
        クラス名　変数名 = new クラス名();

って感じかな。
