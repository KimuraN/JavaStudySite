流れを止めるbreak

breakを用いると流れを止めることができる。whileとかforで繰り返してる途中にもう止めろ！って感じでね。

break;を記述するとその時点でブロックを抜けるってことだ。

            for(int i = 0; i < 10; i++){
                System.out.println(i);
                if(i == 3){
                    break;
                }
            }

上の例をみてほしい。本来なら、10回処理が繰り返されるよね。

しかし、iが3になった時点でbreakが処理される。

breakはブロックを抜けるんだったよね。 そのため繰り返しは4回で終了するよ。

if文のブロック内にbreakがあるから抜けるのはif文のブロックであって for文のブロックは抜けないじゃないかって？？

breakは流れを止めるんだよ。if文に流れはないよね。だからfor文のブロックを 抜けるよ。したがってfor文やwhile文を使わずにif文とbreak文の二つで使うこともできないよ。 

            int a = 3;
            if(a < 3){
                break;
            }

例

        class Test{
            public static void main(String[] args){
                for(int i = 0; i < 5; i++){
                    for(int j = 0; j < 5; j++){
                        System.out.println("j");
                        if( j==2 ){
                            break;
                        }
                    }
                    System.out.println("i");
                }
            }
        }

ループのなかにループがある二重ループだね。内側のループでbreakしているから、内側のfor文は抜けるけど 外側のfor文は抜けないよ。 

        class Test{
            public static void main(String[] args){
                    int a = 1;
                    switch(a){
                        case:0
                            System.out.println(0);
                            break;
                        case:1
                            System.out.println(1);
                        case:2
                            System.out.println(2);
                            break;
                    }
            }
        }

1を出力した後にbreakがないからブロックを抜けずに2も出力する。
