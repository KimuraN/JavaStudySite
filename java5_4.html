「振る舞い」があるクラス



5-3で作成したクラスは

        class Dog{
            String name;
            int age;
        }

だったね。今回はこのクラスに振る舞いを付け加えよう

まず、犬に必要な振る舞いを考える。

  • 吠える

  • 走る

これぐらいかな。

振る舞いはメソッドにより定義を行う。メソッドはCとかでいう関数みたいなものだよ。

まず、メソッドの構文の説明を行う。

        修飾子 返り値の型 メソッド名(引数){
            メソッドの中身(処理)
        }

以上の様な構文でメソッドを記述できる。

意味が分からないと思うけど、とりあえず今は修飾子はpublic、 返り値の型はvoid,引数は空欄(何も書かない)としておこう。

実際に、吠える、走るのメソッドを記述すると

        class Dog{
            String name;
            int age;
        
            public void bark(){
                System.out.println("wan wan");
            }
            
            public void run(){
                System.out.println("running...");
            }
        }

こうなる。実際にPCから犬の鳴き声を発声させたりするのは 大変だからどちらも吠える走るの処理は文字列を出力するのみにしておいたよ。

おめでとう。これでとりあえず犬のクラスは完成。次回は犬を生成する。
