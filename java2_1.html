変数

C言語でもやったからわかりやすいと思うよ

変数ってなんだ

カンタンにいうと変数とは箱だよ。

箱には何か値を入れておくことができる。

箱に入っているものを取り出すことができる。

箱に入っているものを変更できる。

こんなイメージを持てば良いと思う。



型

変数には値を入れることができるんだけど、その値には色々な種類がある。

数値だったり、文字列だったりね。

値の種類を表すものが型だと考えれば良いよ。以下にいくつか型の例を示しておくね。

  • int　　整数

  • double　　小数点数

  • char　　　文字(一文字)

  • String　　文字列

  • boolean　　trueかfalse(真か偽)

これらの型は基本型というよ(ただしStringは基本型っぽく使えるだけで別物。これだけ大文字で始まってるしね。)。

基本型には他にも色々あるから気になったら調べてみてね。



変数を用意する

そろそろ変数を使ってみるよ。

変数を使うにはまず変数を用意しなくてはいけない。箱もなにもないのに値を入れたりすることはできないからね。

変数を用意するには以下のようにコードを書くよ。

		型名 変数名;

このように、変数に入れる値の種類(型)と変数の名前を指定して変数を用意するよ。以下に例を示すね。

		class Test{
			public static void main(String[] args){
				int number;
			}
		}

上の例では、int型の値を入れることができるnumberって名前の箱を用意したって考えればいいよ。

ちなみに変数を用意することを宣言するっていうことが多いから覚えておいて。



変数に値を入れる

変数を用意したらそこに何か値を入れよう。用意しただけでは何も入ってないからね。

ちなみに値をいれることは代入ということが多いよ。

			class Test{
				public static void main(String[] args){
					int number;
				}
			}

上の様に宣言したnumberという変数に値を入れてみる。

変数に値をいれるときは、以下の様にコードを書くよ。

		変数名 = 値;

つまり、

			class Test{
				public static void main(String[] args){
					int number;
					number = 4649;
				}
			}

この様にして、numberに値(4649)を代入できるよ。イコールには右辺と左辺の値が同じってイメージがあるけど、 プログラミング的には右辺のものを左辺のものに入れるって感じだから注意しておいてね。

変数の宣言と代入を一気に、

			class Test{
				public static void main(String[] args){
					char c = 'A';	
				}
			}

としてしまうこともできるよ。(文字(一文字)は'と'で囲む。)

			class Test{
				public static void main(String[] args){
					char c,c2,c3,c4;	
				}
			}

こんな感じに一度にたくさんの同じ型の変数を宣言することもできるよ。

注意してほしいけど、ある型で宣言された変数に対して、他の型の値を代入することは基本的にできないよ。

例えばint型で宣言した変数に、double型の値を入れるみたいな感じね。

			class Test{
				public static void main(String[] args){
					int i = 3.44;
				}
			}

これを許したら、何のために型を指定して変数を宣言したんだってなるでしょ。



変数にいれる値は1つだけ

変数にはたくさんの値をいれておくことはできないよ。

			class Test{
				public static void main(String[] args){
					int i;
					i = 3;
					i = 5;
				}
			}

上の例ではiに3を代入して、その後に5を代入してるけど、iに3と5の両方の値を入れておくことはできないよ。

iに3を代入した時点ではiには3が入っているけど、i=5をした時点で3は消えてiには5が入っていることになるよ。



変数から値を取り出す

変数に値をいれっぱなしだとあれだから、入れた値は使いたいよね。

変数から値を取り出すような文法はないよ。(代入にはイコールがあったね。)

ただ、変数を値の様に扱えばいいんだよ。例をみたほうが分かりやすいと思うから以下にいくつか例を上げるね。

			class Test{
				public static void main(String[] args){
					int i;
					int num;
					char c = 'A';
					i = 3;
					num = i;
					System.out.println(i);
					i = 5;
					System.out.println(i);
					System.out.println(num);
					System.out.println(c);
				}
			}
		
			3
			5
			3
			A

			class Test{
				public static void main(String[] args){
					int i = 10;
					i+3;
					System.out.println(i);
					i = i+10;
					System.out.println(i);
				}
			}
		
			10
			20

i+3をしてるけど、その結果をどこにもいれていないからiの中身は10のまま。i=i+10では、i+10の結果をiに代入してるから iの中身は20になる。


			class Test{
				public static void main(String[] args){
					String str = "banana";
					int i = 3;
					System.out.println(i+10);
					System.out.println(i+str);
				}
			}
		
			13
			3banana

上に示した例からどんな感じかわかってくれたかな。この変は慣れれば簡単だからいろいろいじってみれば良いと思うよ。



おまけ

			i = i+3;
			num = num+10;
			str = str+"hello";

この様な文は、

			i += 3;
			num += 10;
			str += "hello";

と書くことができるよ。(+の部分は-とか*とか変えてもいいからね)（文字列に対して-とか*は使えないとかは気をつけてね。）
