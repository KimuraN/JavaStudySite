「状態」があるクラス



5-2で作成したクラスは

        class Dog{
        
        }

だったよね。これは状態も振る舞いもない。

今回はこのクラスに状態を付け加えよう。

状態のことはフィールドとかフィールド変数とか、インスタンス変数などと呼ばれるよ。

とりあえずは、クラスが持っている変数が状態だと思えばいいと思うよ。

まず、犬に必要な状態を考える。

  • 名前

  • 年齢

とりあえず、これぐらいにしよう。それじゃあこれらの状態をクラスに定義するよ。

どう定義するかって？？。クラスにそれぞれの変数を宣言すれば良い。

        class Dog{
            String name;
            int age;
        }

こんな感じだよ。

この名前と年齢を持った犬を生成することができるクラスを定義することができた。振る舞い は何もないから何をすることもできないけどね。

ちなみに、いままで変数はmainメソッドの中で宣言してきたのに対し、Dogクラスはmainメソッド が存在さえしない。

クラスは設計図だと説明したけど、設計図は何かを実行するものではないよね。

したがってmainメソッドは必要ないよ。

じゃあどこで犬を生成し実行するかということはのちのち説明する。
