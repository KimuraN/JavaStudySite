for文

for文は同じ処理を複数回繰り返す時に使うことができる。

例えば「こんにちは」を2000回出力したいとしよう。

君は以下の様なコードを書くかもしれないね。

		class Test {
			public static void main(String[] args) {
			        System.out.prinltn("こんにちは");
			        System.out.prinltn("こんにちは");
			        System.out.prinltn("こんにちは");
			        System.out.prinltn("こんにちは");
			        ・・・・
			        System.out.prinltn("こんにちは");
			}
		}

たしかにSystem.out.println("こんにちは")を2000回書けば実現可能だ。

でもそんなのは面倒だよね。ここでfor文の出番だね。for文を使うと以下の様に実現できる。 

		class Test {
			public static void main(String[] args) {
                for(int i = 0; i < 2000; i++){
			        System.out.prinltn("こんにちは");
				}
			}
		}

どうかな。文法的意味は分からなくても行数が減ったことは分かるよね。

それじゃあ、for文の構文を説明する。

            for(　初期処理; 条件; 繰り返しが一度終わった後の処理){
	            繰り返す処理
			}

詳しい説明を例1を用いてしていくね。

初期処理は、繰り返し文を行う前に一度だけ処理される。

例1では変数iを用意して0で初期化しているね。

条件は、if文の時に出てきた条件と同じ。条件が真である限り繰り返しは続くよ。

繰り返しが一度終わった後の処理は文字通り一度そのまま。

例1では、繰り返し一回毎にiの値を1増加させている。



別に書かなくてもいいよ

for文には必ず、初期処理、条件、繰り返し後の処理、が必要というわけではない。

例えば初期処理がないfor文

            for(　; 条件; 繰り返しが一度終わった後の処理){
	            繰り返す処理
			}

であったり、初期処理と条件がないfor文

            for(　;　; 繰り返しが一度終わった後の処理){
	            繰り返す処理(条件を書かない場合は条件が真となる。)
			}

であったり全てがないfor文

            for(; ;){
	            繰り返す処理
			}

などが可能だ。セミコロンは必要だから忘れないでね。



変数の有効範囲

まずは下の例を見てほしい。

            for(int a = 10; a<20; a++){
	            System.out.println(a);
			}

変数aを用意してaの値を増加させながら出力するfor文だね。

今回はこのaに注目してほしい。基本的に変数は宣言したブロック内でしか 使うことができない。何度もいうけどブロックとは{}で囲まれた部分だよ。

aはfor文内で宣言したからfor文のブロック内でしか使用ができない。

つまり、下の様なのはダメってこと。

        class Test{
            public static void main(String[] args){
                for(int a = 0; a<20; a++){
                    System.out.println(a);
                }
                System.out.prinltn(a);//for文の外でaは使えない。
            }
        }

変数の有効範囲はスコープと呼ばれている。上で述べた話はfor文時に限った ものではないから、時間があればのちのちさらに詳しく説明するよ。



例

        class Test{
            public static void main(String[] args){
                int a = 3;
                for(int a = 20; a<30; a++){
                    System.out.println(a);
                }
            }
        } 
		20から30を出力
        class Test{
            public static void main(String[] args){
                for(int a = 20; a>0; a--){
                    System.out.println(a);
                }
            }
        } 
		20から1を出力
