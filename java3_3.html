for文

for文は同じ処理を複数回繰り返す時に使うことができる。

例えば「こんにちは」を2000回出力したいとしよう。

君は以下の様なコードを書くかもしれないね。

		class Test {
			public static void main(String[] args) {
			        System.out.prinltn("こんにちは");
			        System.out.prinltn("こんにちは");
			        System.out.prinltn("こんにちは");
			        System.out.prinltn("こんにちは");
			        ・・・・
			        System.out.prinltn("こんにちは");
			}
		}

たしかにSystem.out.println("こんにちは")を2000回書けば実現可能だ。

でもそんなのは面倒だよね。ここでfor文の出番だね。for文を使うと以下の様に実現できる。 

		class Test {
			public static void main(String[] args) {
                for(int i = 0; i < 2000; i++){
			        System.out.prinltn("こんにちは");
				}
			}
		}

どうかな。文法的意味は分からなくても行数が減ったことは分かるよね。

それじゃあ、for文の構文を説明する。
