if文

例えば、「明日晴れたら外へ出て公園へ行く」というのは良くあるようなことだけど、

これは条件「明日が晴れ」によって行動を選択しているね。

このようにある条件の結果に応じて処理を行うことがif文を使うとできるようになるよ。

if文の書き方は以下の様な感じ。

			if(条件){
				条件が真だった時の処理をこのブロックに書く。
			}

上で述べた日本語の例で当てはめてみると、

			if(明日晴れる){
				外へ行く;
				公園へ行く;
			}

こんな感じ。



if文の条件の部分にはtrueかfalse

上で示したif分の条件の部分にはtrueかfalseが得られる文を書けばいいよ。

条件がtrueの時のみ後ろのブロック内に書いた処理が行われるよ。

ちょっと例を示すね。

			class Test{
				public static void main(String[] args){
		
					if(true){
						System.out.println("いえい");
						Ststem.out.println("ふふふ");
					}
					System.out.println("どうだったかな");
					
				}
			}
		
			いえい
			ふふふ
			どうだったかな
			class Test{
				public static void main(String[] args){

					if(false){
						System.out.println("いえい");
						Ststem.out.println("ふふふ");
					}
					System.out.println("どうだったかな");
				}
			}
		
			どうだったかな

ここで思い出してほしいのはboolaenのことだ。

booleanの型の変数にはtrueとfalseが代入できたね。でも、だからといってboolean型の変数に対して

		boolean b = true;
		boolean b = false;

しかできなかったというとそれは違うね。

		boolean b = 1<3;
		boolean b = (3==3) && (3!=3);

こんなこともできたよね。覚えてるかな。そして！！if文でもこれは同様なんだよ！！以下の様なことができるんだよ。

そしてこっちの方が一般的だよ。条件にtrueやfalseを直接書くなんてif文使う意味あるのかよって感じだしね。

			class Test{
				public static void main(String[] args){
					int i = 0;
					
					if( i<10 ){
						System.out.println("iの値は10より小さい");
					}
					System.out.println("どうだったかな");
					
				}
			}
		
			iの値は10より小さい
			どうだったかな

そうでなかったら

最初の例「明日晴れたら外へ出て公園へ行く」を思い出してほしい。

これに、そうでなかったらどうするかを追加しみてみよう。

「明日晴れたら外へ出て公園へ行く。そうでなかったら家にいる。」って感じ。

つまり条件が真だったら外出て公園行く。偽だったら家にいるって感じだね。

この様な分岐はif　elseを使うと実現できるよ。書き方は以下の様な感じ。

			if(条件){
				条件が真だった時の処理をこのブロックに書く。
			}else{
				条件が偽だった時の処理をこのブロックに書く。
			}

上で述べた日本語の例で当てはめてみると、

			if(明日晴れる){
				外へ行く;
				公園へ行く;
			}else{
				家にいる;
			}

こんな感じ。例をあげておくね。

			class Test{
				public static void main(String[] args){
					int a = 10;
					if( a<20 ){
						System.out.println("aの値は20より小さい");
					}else{
						System.out.println("aの値は20より大きい");
					}
					
					System.out.println("どうだったかな");
					
				}
			}
		
			aの値は20より小さい
			どうだったかな


複数の条件

今度は最初の例「明日晴れたら外へ出て公園へ行く」に複数の条件を追加してみよう。

「明日晴れたら外へ出て公園へ行く、晴れでなく曇りだったら図書館へ行く、晴れでも曇りでもなく雨だったら家にいる、どれでもなかったら何もしない」

この様な複数の条件の判定を行う場合は以下の様に書く。

			if(条件1){
				条件1が真だった時の処理をこのブロックに書く。
			}else if(条件2){
				条件1が偽で条件2が真だった時の処理をこのブロックに書く。
			}else if(条件3){
				条件1と条件2が偽で条件3が真だった時の処理をこのブロックに書く。
			}else{
				どの条件も偽だった時の処理をこのブロックに書く。
			}

上で述べた日本語の例で当てはめてみると、

			if(明日晴れる){
				外へ行く
				公園へ行く
			}else if(明日曇る){
				図書館へ行く
			}else if(明日雨){
				家にいる。
			}else{
				何もしない。
			}

else ifはいくつでも書くことができるよ。上から順に条件を判定してtrueとなった条件部分の処理を行うよ。

ただし、trueとなった条件が見つかりその部分の処理を行ったらif文はそこで終わってその先の判定は行わないよ。

どういうことかというと、明日晴れるが真だったから外行く、公園行く処理を行った、この後に明日曇りだったり明日雨という条件の判定は行わないってこと。

		class Test{
			public static void main(String[] args){
    			int i = 20;
	    		if(i > 100){
	    			System.out.println("100より大きい");
	    		}else if(i > 50){
	    			System.out.println("50～100");
	    		}else if(i > 0){
	    			System.out.println("0～50");
	    		}else{
	    			System.out.println("0以下");
	    		}
			}
		}	
		
			0～50

if文の中にif文。(if文のネストというよ。入れ子ともいうよ。)

		class Test{
			public static void main(String[] args){
				int i = 20;
    			if(i != 100){
    				if(i < 50){
    					System.out.println("100でなく50以下");
    				}
    			}
			}
		}
		
			100でなく50以下

これは下の様に書き換えが可能。

	
		class Test{
   			public static void main(String[] args){
    			int i = 20;
    			if( (i != 100) && (i<50) ){
    				System.out.println("100でなく50以下");
    			}
			}
		}
		
			100でなく50以下
